- 로마자 외에 다른 문자를 섞어 쓸 때, ASCII 인코딩 오류가 날 때, 위 내용을 코드 맨 위에 추가
- `#`가 주석에 쓰이는 문자인데, 위 코드는 어떻게 동작하는 것일까?
    - 파이썬은 윗 줄도 코드로 인식하지 않는다.
    - 파일의 포맷 설정을 찾아내는 `해킹`이나 대안해법(workaround) 가운데 한 방법으로 사용함.

#### `=` 와 `==`의 차이점
- `=` → 오른쪽의 값을 왼쪽의 변수에 대입
- `==` → 양쪽의 값이 같은지 비교
